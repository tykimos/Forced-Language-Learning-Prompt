# Language Echo Rule (French → Polish)

> For **French** (Français) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in French.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in French.
