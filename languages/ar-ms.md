# Language Echo Rule (Arabic → Malay)

> For **Arabic** (العربية) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Arabic.
