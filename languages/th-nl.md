# Language Echo Rule (Thai → Dutch)

> For **Thai** (ภาษาไทย) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Thai.
