# Language Echo Rule (Vietnamese → Malay)

> For **Vietnamese** (Tiếng Việt) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Vietnamese.
