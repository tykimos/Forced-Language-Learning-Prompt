# Language Echo Rule (Indonesian → Bengali)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Indonesian.
