# Language Echo Rule (Italian → Russian)

> For **Italian** (Italiano) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Italian.
