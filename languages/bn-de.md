# Language Echo Rule (Bengali → German)

> For **Bengali** (বাংলা) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Bengali.
