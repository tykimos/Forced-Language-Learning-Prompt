# Language Echo Rule (Russian → Bengali)

> For **Russian** (Русский) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Russian.
