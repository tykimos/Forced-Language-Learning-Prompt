# Language Echo Rule (Russian → Korean)

> For **Russian** (Русский) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Russian.
