# Language Echo Rule (Russian → Portuguese)

> For **Russian** (Русский) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Russian.
