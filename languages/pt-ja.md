# Language Echo Rule (Portuguese → Japanese)

> For **Portuguese** (Português) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Portuguese.
