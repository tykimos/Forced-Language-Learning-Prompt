# Language Echo Rule (Portuguese → Indonesian)

> For **Portuguese** (Português) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Portuguese.
