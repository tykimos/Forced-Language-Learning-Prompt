# Language Echo Rule (Swedish → Vietnamese)

> For **Swedish** (Svenska) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Swedish.
