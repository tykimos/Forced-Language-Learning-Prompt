# Language Echo Rule (French → Turkish)

> For **French** (Français) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in French.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in French.
