# Language Echo Rule (German → Ukrainian)

> For **German** (Deutsch) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in German.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in German.
