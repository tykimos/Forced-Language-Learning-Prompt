# Language Echo Rule (Dutch → Indonesian)

> For **Dutch** (Nederlands) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Dutch.
