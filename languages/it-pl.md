# Language Echo Rule (Italian → Polish)

> For **Italian** (Italiano) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Italian.
