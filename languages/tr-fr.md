# Language Echo Rule (Turkish → French)

> For **Turkish** (Türkçe) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Turkish.
