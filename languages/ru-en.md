# Language Echo Rule (Russian → English)

> For **Russian** (Русский) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Russian.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Russian.
