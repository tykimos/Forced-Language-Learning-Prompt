# Language Echo Rule (Swedish → Malay)

> For **Swedish** (Svenska) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Swedish.
