# Language Echo Rule (Arabic → Filipino)

> For **Arabic** (العربية) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Arabic.
