# Language Echo Rule (Indonesian → German)

> For **Indonesian** (Bahasa Indonesia) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Indonesian.
