# Language Echo Rule (Spanish → Polish)

> For **Spanish** (Español) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Spanish**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Spanish.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Spanish.
