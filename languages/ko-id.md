# Language Echo Rule (Korean → Indonesian)

> For **Korean** (한국어) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Korean.
