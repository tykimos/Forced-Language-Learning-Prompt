# Language Echo Rule (Swedish → Turkish)

> For **Swedish** (Svenska) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Swedish.
