# Language Echo Rule (Indonesian → Dutch)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Indonesian.
