# Language Echo Rule (Japanese → Polish)

> For **Japanese** (日本語) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Japanese.
