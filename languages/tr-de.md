# Language Echo Rule (Turkish → German)

> For **Turkish** (Türkçe) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Turkish.
