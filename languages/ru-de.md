# Language Echo Rule (Russian → German)

> For **Russian** (Русский) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Russian.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Russian.
