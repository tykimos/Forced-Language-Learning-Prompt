# Language Echo Rule (Dutch → French)

> For **Dutch** (Nederlands) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Dutch.
