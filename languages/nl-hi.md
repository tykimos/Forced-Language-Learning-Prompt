# Language Echo Rule (Dutch → Hindi)

> For **Dutch** (Nederlands) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Dutch.
