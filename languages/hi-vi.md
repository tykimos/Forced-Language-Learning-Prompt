# Language Echo Rule (Hindi → Vietnamese)

> For **Hindi** (हिन्दी) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Hindi.
