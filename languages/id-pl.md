# Language Echo Rule (Indonesian → Polish)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Indonesian.
