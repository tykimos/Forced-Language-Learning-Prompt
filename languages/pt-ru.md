# Language Echo Rule (Portuguese → Russian)

> For **Portuguese** (Português) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Portuguese.
