# Language Echo Rule (Japanese → Vietnamese)

> For **Japanese** (日本語) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Japanese.
