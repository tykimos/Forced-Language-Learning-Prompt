# Language Echo Rule (Portuguese → Malay)

> For **Portuguese** (Português) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Portuguese.
