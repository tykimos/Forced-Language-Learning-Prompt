# Language Echo Rule (Thai → Arabic)

> For **Thai** (ภาษาไทย) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Thai.
