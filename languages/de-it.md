# Language Echo Rule (German → Italian)

> For **German** (Deutsch) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in German.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in German.
