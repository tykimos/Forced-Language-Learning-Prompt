# Language Echo Rule (Vietnamese → Polish)

> For **Vietnamese** (Tiếng Việt) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Vietnamese.
