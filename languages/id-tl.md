# Language Echo Rule (Indonesian → Filipino)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Indonesian.
