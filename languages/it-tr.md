# Language Echo Rule (Italian → Turkish)

> For **Italian** (Italiano) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Italian.
