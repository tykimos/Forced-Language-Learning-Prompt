# Language Echo Rule (Filipino → Bengali)

> For **Filipino** (Filipino) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Filipino.
