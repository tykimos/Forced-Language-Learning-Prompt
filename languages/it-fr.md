# Language Echo Rule (Italian → French)

> For **Italian** (Italiano) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Italian.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Italian.
