# Language Echo Rule (Vietnamese → Thai)

> For **Vietnamese** (Tiếng Việt) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Vietnamese.
