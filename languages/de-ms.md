# Language Echo Rule (German → Malay)

> For **German** (Deutsch) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in German.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in German.
