# Language Echo Rule (Arabic → Italian)

> For **Arabic** (العربية) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Arabic.
