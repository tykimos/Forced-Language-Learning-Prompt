# Language Echo Rule (Italian → German)

> For **Italian** (Italiano) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Italian.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Italian.
