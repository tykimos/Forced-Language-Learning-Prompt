# Language Echo Rule (Portuguese → Italian)

> For **Portuguese** (Português) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Portuguese.
