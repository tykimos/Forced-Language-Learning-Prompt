# Language Echo Rule (Filipino → Portuguese)

> For **Filipino** (Filipino) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Filipino.
