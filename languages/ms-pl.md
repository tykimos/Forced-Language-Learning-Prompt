# Language Echo Rule (Malay → Polish)

> For **Malay** (Bahasa Melayu) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Malay.
