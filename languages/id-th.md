# Language Echo Rule (Indonesian → Thai)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Indonesian.
