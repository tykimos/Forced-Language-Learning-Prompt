# Language Echo Rule (Korean → Dutch)

> For **Korean** (한국어) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Korean.
