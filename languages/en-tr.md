# Language Echo Rule (English → Turkish)

> For **English** (English) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in English.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in English.
