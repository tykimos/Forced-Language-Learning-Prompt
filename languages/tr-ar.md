# Language Echo Rule (Turkish → Arabic)

> For **Turkish** (Türkçe) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Turkish.
