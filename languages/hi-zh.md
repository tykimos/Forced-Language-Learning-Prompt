# Language Echo Rule (Hindi → Chinese)

> For **Hindi** (हिन्दी) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Hindi.
