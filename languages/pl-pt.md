# Language Echo Rule (Polish → Portuguese)

> For **Polish** (Polski) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Polish.
