# Language Echo Rule (French → Italian)

> For **French** (Français) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in French.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in French.
