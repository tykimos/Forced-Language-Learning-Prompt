# Language Echo Rule (Turkish → Ukrainian)

> For **Turkish** (Türkçe) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Turkish.
