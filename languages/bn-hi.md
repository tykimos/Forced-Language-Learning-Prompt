# Language Echo Rule (Bengali → Hindi)

> For **Bengali** (বাংলা) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Bengali.
