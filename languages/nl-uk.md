# Language Echo Rule (Dutch → Ukrainian)

> For **Dutch** (Nederlands) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Dutch.
