# Language Echo Rule (Hindi → German)

> For **Hindi** (हिन्दी) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Hindi.
