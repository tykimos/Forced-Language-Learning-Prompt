# Language Echo Rule (Bengali → Thai)

> For **Bengali** (বাংলা) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Bengali.
