# Language Echo Rule (Indonesian → Arabic)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Indonesian.
