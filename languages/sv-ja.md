# Language Echo Rule (Swedish → Japanese)

> For **Swedish** (Svenska) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Swedish.
