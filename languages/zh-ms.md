# Language Echo Rule (Chinese → Malay)

> For **Chinese** (中文) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Chinese.
