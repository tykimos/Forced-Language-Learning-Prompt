# Language Echo Rule (Vietnamese → Swedish)

> For **Vietnamese** (Tiếng Việt) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Vietnamese.
