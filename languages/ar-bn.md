# Language Echo Rule (Arabic → Bengali)

> For **Arabic** (العربية) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Arabic.
