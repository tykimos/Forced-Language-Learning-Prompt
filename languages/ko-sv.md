# Language Echo Rule (Korean → Swedish)

> For **Korean** (한국어) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Korean.
