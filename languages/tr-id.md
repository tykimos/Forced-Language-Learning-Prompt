# Language Echo Rule (Turkish → Indonesian)

> For **Turkish** (Türkçe) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Turkish.
