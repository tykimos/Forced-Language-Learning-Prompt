# Language Echo Rule (Ukrainian → Russian)

> For **Ukrainian** (Українська) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Ukrainian.
