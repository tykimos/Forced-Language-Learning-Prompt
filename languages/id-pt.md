# Language Echo Rule (Indonesian → Portuguese)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Indonesian.
