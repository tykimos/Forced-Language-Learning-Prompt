# Language Echo Rule (Hindi → Thai)

> For **Hindi** (हिन्दी) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Hindi.
