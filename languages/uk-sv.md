# Language Echo Rule (Ukrainian → Swedish)

> For **Ukrainian** (Українська) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Ukrainian.
