# Language Echo Rule (Portuguese → Arabic)

> For **Portuguese** (Português) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Portuguese.
