# Language Echo Rule (Portuguese → Chinese)

> For **Portuguese** (Português) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Portuguese.
