# Language Echo Rule (Portuguese → Thai)

> For **Portuguese** (Português) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Portuguese.
