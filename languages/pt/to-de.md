# Language Echo Rule (Portuguese → German)

> For **Portuguese** (Português) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Portuguese.
