# Language Echo Rule (Korean → Russian)

> For **Korean** (한국어) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Korean.
