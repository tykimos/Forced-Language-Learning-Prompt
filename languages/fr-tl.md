# Language Echo Rule (French → Filipino)

> For **French** (Français) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in French.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in French.
