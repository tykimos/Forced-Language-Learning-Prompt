# Language Echo Rule (Filipino → Chinese)

> For **Filipino** (Filipino) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Filipino.
