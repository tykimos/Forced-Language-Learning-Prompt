# Language Echo Rule (Bengali → French)

> For **Bengali** (বাংলা) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Bengali.
