# Language Echo Rule (Italian → Hindi)

> For **Italian** (Italiano) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Italian.
