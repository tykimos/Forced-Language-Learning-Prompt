# Language Echo Rule (Dutch → Thai)

> For **Dutch** (Nederlands) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Dutch.
