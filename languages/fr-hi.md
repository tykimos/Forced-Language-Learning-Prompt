# Language Echo Rule (French → Hindi)

> For **French** (Français) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in French.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in French.
