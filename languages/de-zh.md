# Language Echo Rule (German → Chinese)

> For **German** (Deutsch) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in German.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in German.
