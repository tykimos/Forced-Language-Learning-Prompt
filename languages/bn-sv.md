# Language Echo Rule (Bengali → Swedish)

> For **Bengali** (বাংলা) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Bengali.
