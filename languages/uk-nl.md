# Language Echo Rule (Ukrainian → Dutch)

> For **Ukrainian** (Українська) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Ukrainian.
