# Language Echo Rule (Bengali → Ukrainian)

> For **Bengali** (বাংলা) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Bengali.
