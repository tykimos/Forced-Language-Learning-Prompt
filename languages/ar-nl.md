# Language Echo Rule (Arabic → Dutch)

> For **Arabic** (العربية) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Arabic.
