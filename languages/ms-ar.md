# Language Echo Rule (Malay → Arabic)

> For **Malay** (Bahasa Melayu) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Malay.
