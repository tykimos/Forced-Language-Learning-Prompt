# Language Echo Rule (Italian → English)

> For **Italian** (Italiano) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Italian.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Italian.
