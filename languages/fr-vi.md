# Language Echo Rule (French → Vietnamese)

> For **French** (Français) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in French.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in French.
