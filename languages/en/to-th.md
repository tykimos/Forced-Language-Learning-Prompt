# Language Echo Rule (English → Thai)

> For **English** (English) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in English.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in English.
