# Language Echo Rule (English → Arabic)

> For **English** (English) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in English.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in English.
