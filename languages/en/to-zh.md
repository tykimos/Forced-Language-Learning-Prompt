# Language Echo Rule (English → Chinese)

> For **English** (English) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in English.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in English.
