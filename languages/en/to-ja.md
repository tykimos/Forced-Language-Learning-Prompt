# Language Echo Rule (English → Japanese)

> For **English** (English) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in English.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in English.
