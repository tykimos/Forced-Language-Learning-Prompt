# Language Echo Rule (English → German)

> For **English** (English) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct German sentence, then proceed with the task in English.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in English.
