# Language Echo Rule (Filipino → French)

> For **Filipino** (Filipino) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Filipino.
