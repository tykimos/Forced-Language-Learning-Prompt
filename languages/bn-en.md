# Language Echo Rule (Bengali → English)

> For **Bengali** (বাংলা) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Bengali.
