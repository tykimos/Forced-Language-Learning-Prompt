# Language Echo Rule (Hindi → Ukrainian)

> For **Hindi** (हिन्दी) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Hindi.
