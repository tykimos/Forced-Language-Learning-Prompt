# Language Echo Rule (Russian → Hindi)

> For **Russian** (Русский) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Russian.
