# Language Echo Rule (Chinese → Indonesian)

> For **Chinese** (中文) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Chinese.
