# Language Echo Rule (Bengali → Vietnamese)

> For **Bengali** (বাংলা) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Bengali.
