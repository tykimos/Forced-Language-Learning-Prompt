# Language Echo Rule (Polish → Russian)

> For **Polish** (Polski) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Polish.
