# Language Echo Rule (Dutch → Spanish)

> For **Dutch** (Nederlands) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Dutch.
