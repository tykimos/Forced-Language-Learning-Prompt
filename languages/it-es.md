# Language Echo Rule (Italian → Spanish)

> For **Italian** (Italiano) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Italian.
