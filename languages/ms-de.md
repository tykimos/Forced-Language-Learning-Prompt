# Language Echo Rule (Malay → German)

> For **Malay** (Bahasa Melayu) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Malay.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Malay.
