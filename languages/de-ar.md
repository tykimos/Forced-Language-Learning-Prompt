# Language Echo Rule (German → Arabic)

> For **German** (Deutsch) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in German.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in German.
