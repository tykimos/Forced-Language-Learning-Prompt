# Language Echo Rule (Hindi → Polish)

> For **Hindi** (हिन्दी) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Hindi.
