# Language Echo Rule (Arabic → Turkish)

> For **Arabic** (العربية) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Arabic.
