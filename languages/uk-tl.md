# Language Echo Rule (Ukrainian → Filipino)

> For **Ukrainian** (Українська) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Ukrainian.
