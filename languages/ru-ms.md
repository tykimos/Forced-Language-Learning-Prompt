# Language Echo Rule (Russian → Malay)

> For **Russian** (Русский) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Russian.
