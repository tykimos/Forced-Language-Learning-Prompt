# Language Echo Rule (Russian → Indonesian)

> For **Russian** (Русский) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Russian.
