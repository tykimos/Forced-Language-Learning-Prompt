# Language Echo Rule (Indonesian → Ukrainian)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Indonesian.
