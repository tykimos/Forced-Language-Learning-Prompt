# Language Echo Rule (Italian → Malay)

> For **Italian** (Italiano) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Italian.
