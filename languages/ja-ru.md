# Language Echo Rule (Japanese → Russian)

> For **Japanese** (日本語) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Japanese.
