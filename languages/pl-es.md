# Language Echo Rule (Polish → Spanish)

> For **Polish** (Polski) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Polish.
