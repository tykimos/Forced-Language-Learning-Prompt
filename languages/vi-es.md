# Language Echo Rule (Vietnamese → Spanish)

> For **Vietnamese** (Tiếng Việt) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Vietnamese.
