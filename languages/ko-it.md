# Language Echo Rule (Korean → Italian)

> For **Korean** (한국어) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Korean.
