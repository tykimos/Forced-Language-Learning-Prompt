# Language Echo Rule (Italian → Arabic)

> For **Italian** (Italiano) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Italian.
