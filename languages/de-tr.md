# Language Echo Rule (German → Turkish)

> For **German** (Deutsch) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in German.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in German.
