# Language Echo Rule (Turkish → Swedish)

> For **Turkish** (Türkçe) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Turkish.
