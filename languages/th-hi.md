# Language Echo Rule (Thai → Hindi)

> For **Thai** (ภาษาไทย) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Thai.
