# Language Echo Rule (Bengali → Italian)

> For **Bengali** (বাংলা) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Bengali.
