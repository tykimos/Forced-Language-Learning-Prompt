# Language Echo Rule (Ukrainian → Thai)

> For **Ukrainian** (Українська) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Ukrainian.
