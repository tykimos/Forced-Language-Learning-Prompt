# Language Echo Rule (Italian → Ukrainian)

> For **Italian** (Italiano) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Italian.
