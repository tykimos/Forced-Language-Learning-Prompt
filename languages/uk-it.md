# Language Echo Rule (Ukrainian → Italian)

> For **Ukrainian** (Українська) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Ukrainian.
