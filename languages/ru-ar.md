# Language Echo Rule (Russian → Arabic)

> For **Russian** (Русский) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Russian.
