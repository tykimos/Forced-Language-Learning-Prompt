# Language Echo Rule (Spanish → Bengali)

> For **Spanish** (Español) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Spanish**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Spanish.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Spanish.
