# Language Echo Rule (Bengali → Arabic)

> For **Bengali** (বাংলা) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Bengali.
