# Language Echo Rule (Chinese → Arabic)

> For **Chinese** (中文) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Chinese.
