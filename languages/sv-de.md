# Language Echo Rule (Swedish → German)

> For **Swedish** (Svenska) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Swedish.
