# Language Echo Rule (Ukrainian → Polish)

> For **Ukrainian** (Українська) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Ukrainian.
