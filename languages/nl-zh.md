# Language Echo Rule (Dutch → Chinese)

> For **Dutch** (Nederlands) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Dutch.
