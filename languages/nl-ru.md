# Language Echo Rule (Dutch → Russian)

> For **Dutch** (Nederlands) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Dutch.
