# Language Echo Rule (German → Polish)

> For **German** (Deutsch) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in German.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in German.
