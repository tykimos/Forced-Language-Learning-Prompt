# Language Echo Rule (Swedish → Ukrainian)

> For **Swedish** (Svenska) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Swedish.
