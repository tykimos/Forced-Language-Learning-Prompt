# Language Echo Rule (German → Swedish)

> For **German** (Deutsch) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in German.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in German.
