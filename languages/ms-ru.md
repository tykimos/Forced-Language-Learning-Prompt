# Language Echo Rule (Malay → Russian)

> For **Malay** (Bahasa Melayu) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Malay.
