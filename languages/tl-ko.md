# Language Echo Rule (Filipino → Korean)

> For **Filipino** (Filipino) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Filipino.
