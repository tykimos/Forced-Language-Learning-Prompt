# Language Echo Rule (Turkish → Bengali)

> For **Turkish** (Türkçe) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Turkish.
