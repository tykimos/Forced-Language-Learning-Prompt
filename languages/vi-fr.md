# Language Echo Rule (Vietnamese → French)

> For **Vietnamese** (Tiếng Việt) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Vietnamese.
