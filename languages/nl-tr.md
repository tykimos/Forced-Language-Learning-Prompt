# Language Echo Rule (Dutch → Turkish)

> For **Dutch** (Nederlands) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Dutch.
