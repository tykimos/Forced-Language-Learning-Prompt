# Language Echo Rule (Hindi → Malay)

> For **Hindi** (हिन्दी) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Hindi.
