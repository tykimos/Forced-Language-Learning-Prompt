# Language Echo Rule (Filipino → Turkish)

> For **Filipino** (Filipino) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Filipino.
