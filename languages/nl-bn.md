# Language Echo Rule (Dutch → Bengali)

> For **Dutch** (Nederlands) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Dutch.
