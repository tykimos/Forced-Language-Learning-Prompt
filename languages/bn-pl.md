# Language Echo Rule (Bengali → Polish)

> For **Bengali** (বাংলা) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Bengali.
