# Language Echo Rule (Bengali → Malay)

> For **Bengali** (বাংলা) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Bengali.
