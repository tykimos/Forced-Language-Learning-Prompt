# Language Echo Rule (Russian → Thai)

> For **Russian** (Русский) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Russian.
