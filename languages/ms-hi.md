# Language Echo Rule (Malay → Hindi)

> For **Malay** (Bahasa Melayu) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Malay.
