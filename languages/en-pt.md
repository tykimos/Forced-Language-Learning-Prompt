# Language Echo Rule (English → Portuguese)

> For **English** (English) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in English.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in English.
