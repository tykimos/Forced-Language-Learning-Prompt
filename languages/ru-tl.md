# Language Echo Rule (Russian → Filipino)

> For **Russian** (Русский) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Russian.
