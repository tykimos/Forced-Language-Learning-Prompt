# Language Echo Rule (Japanese → Portuguese)

> For **Japanese** (日本語) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Japanese.
