# Language Echo Rule (Chinese → Filipino)

> For **Chinese** (中文) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Chinese.
