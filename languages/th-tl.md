# Language Echo Rule (Thai → Filipino)

> For **Thai** (ภาษาไทย) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Thai.
