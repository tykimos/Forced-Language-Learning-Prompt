# Language Echo Rule (Bengali → Indonesian)

> For **Bengali** (বাংলা) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Bengali.
