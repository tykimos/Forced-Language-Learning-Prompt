# Language Echo Rule (Turkish → Malay)

> For **Turkish** (Türkçe) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Turkish.
