# Language Echo Rule (Italian → Dutch)

> For **Italian** (Italiano) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Italian.
