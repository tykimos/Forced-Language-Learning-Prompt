# Language Echo Rule (Italian → Indonesian)

> For **Italian** (Italiano) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Italian.
