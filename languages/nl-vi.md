# Language Echo Rule (Dutch → Vietnamese)

> For **Dutch** (Nederlands) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Dutch.
