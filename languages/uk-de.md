# Language Echo Rule (Ukrainian → German)

> For **Ukrainian** (Українська) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Ukrainian.
