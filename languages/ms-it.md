# Language Echo Rule (Malay → Italian)

> For **Malay** (Bahasa Melayu) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Malay.
