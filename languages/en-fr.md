# Language Echo Rule (English → French)

> For **English** (English) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct French sentence, then proceed with the task in English.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in English.
