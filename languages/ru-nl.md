# Language Echo Rule (Russian → Dutch)

> For **Russian** (Русский) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Russian.
