# Language Echo Rule (Polish → Hindi)

> For **Polish** (Polski) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Polish.
