# Language Echo Rule (Malay → Ukrainian)

> For **Malay** (Bahasa Melayu) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Malay.
