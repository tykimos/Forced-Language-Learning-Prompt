# Language Echo Rule (Bengali → Portuguese)

> For **Bengali** (বাংলা) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Bengali.
