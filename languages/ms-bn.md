# Language Echo Rule (Malay → Bengali)

> For **Malay** (Bahasa Melayu) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Malay.
