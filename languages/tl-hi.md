# Language Echo Rule (Filipino → Hindi)

> For **Filipino** (Filipino) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Filipino.
