# Language Echo Rule (Vietnamese → Italian)

> For **Vietnamese** (Tiếng Việt) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Vietnamese.
