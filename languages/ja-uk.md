# Language Echo Rule (Japanese → Ukrainian)

> For **Japanese** (日本語) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Japanese.
