# Language Echo Rule (Bengali → Chinese)

> For **Bengali** (বাংলা) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Bengali.
