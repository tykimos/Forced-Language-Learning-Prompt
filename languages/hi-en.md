# Language Echo Rule (Hindi → English)

> For **Hindi** (हिन्दी) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Hindi.
