# Language Echo Rule (Chinese → English)

> For **Chinese** (中文) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Chinese.
