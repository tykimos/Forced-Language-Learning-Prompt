# Language Echo Rule (Korean → Vietnamese)

> For **Korean** (한국어) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Korean.
