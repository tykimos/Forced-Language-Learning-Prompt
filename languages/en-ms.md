# Language Echo Rule (English → Malay)

> For **English** (English) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in English.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in English.
