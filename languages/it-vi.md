# Language Echo Rule (Italian → Vietnamese)

> For **Italian** (Italiano) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Italian.
