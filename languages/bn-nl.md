# Language Echo Rule (Bengali → Dutch)

> For **Bengali** (বাংলা) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Bengali.
