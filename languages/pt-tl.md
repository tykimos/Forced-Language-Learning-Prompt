# Language Echo Rule (Portuguese → Filipino)

> For **Portuguese** (Português) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Portuguese.
