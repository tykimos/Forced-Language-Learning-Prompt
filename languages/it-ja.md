# Language Echo Rule (Italian → Japanese)

> For **Italian** (Italiano) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Italian.
