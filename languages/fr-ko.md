# Language Echo Rule (French → Korean)

> For **French** (Français) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in French.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in French.
