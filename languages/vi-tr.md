# Language Echo Rule (Vietnamese → Turkish)

> For **Vietnamese** (Tiếng Việt) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Vietnamese.
