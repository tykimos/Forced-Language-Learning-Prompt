# Language Echo Rule (Thai → Turkish)

> For **Thai** (ภาษาไทย) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Thai.
