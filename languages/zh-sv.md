# Language Echo Rule (Chinese → Swedish)

> For **Chinese** (中文) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Chinese.
