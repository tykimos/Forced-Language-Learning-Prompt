# Language Echo Rule (Portuguese → Polish)

> For **Portuguese** (Português) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Portuguese.
