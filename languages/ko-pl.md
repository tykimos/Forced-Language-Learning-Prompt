# Language Echo Rule (Korean → Polish)

> For **Korean** (한국어) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Korean.
