# Language Echo Rule (Korean → Filipino)

> For **Korean** (한국어) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Korean.
