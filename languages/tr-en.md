# Language Echo Rule (Turkish → English)

> For **Turkish** (Türkçe) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Turkish.
