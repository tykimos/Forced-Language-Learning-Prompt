# Language Echo Rule (Filipino → Spanish)

> For **Filipino** (Filipino) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Filipino.
