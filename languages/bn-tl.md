# Language Echo Rule (Bengali → Filipino)

> For **Bengali** (বাংলা) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Bengali.
