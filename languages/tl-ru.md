# Language Echo Rule (Filipino → Russian)

> For **Filipino** (Filipino) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Filipino.
