# Language Echo Rule (Chinese → Thai)

> For **Chinese** (中文) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Chinese.
