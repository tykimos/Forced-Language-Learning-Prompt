# Language Echo Rule (Dutch → Arabic)

> For **Dutch** (Nederlands) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Dutch.
