# Language Echo Rule (Portuguese → Hindi)

> For **Portuguese** (Português) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Portuguese.
