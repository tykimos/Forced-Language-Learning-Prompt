# Language Echo Rule (German → Bengali)

> For **German** (Deutsch) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in German.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in German.
