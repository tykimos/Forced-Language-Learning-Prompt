# Language Echo Rule (Russian → Spanish)

> For **Russian** (Русский) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Russian.
