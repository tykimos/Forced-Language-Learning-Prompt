# Language Echo Rule (Portuguese → Ukrainian)

> For **Portuguese** (Português) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Portuguese.
