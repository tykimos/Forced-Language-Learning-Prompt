# Language Echo Rule (Russian → Vietnamese)

> For **Russian** (Русский) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Russian.
