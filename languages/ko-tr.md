# Language Echo Rule (Korean → Turkish)

> For **Korean** (한국어) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Korean.
