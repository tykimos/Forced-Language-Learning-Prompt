# Language Echo Rule (Arabic → Russian)

> For **Arabic** (العربية) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Arabic.
