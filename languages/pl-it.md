# Language Echo Rule (Polish → Italian)

> For **Polish** (Polski) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Polish.
