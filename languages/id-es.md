# Language Echo Rule (Indonesian → Spanish)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Indonesian.
