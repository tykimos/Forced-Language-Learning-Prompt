# Language Echo Rule (Portuguese → Vietnamese)

> For **Portuguese** (Português) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Portuguese.
