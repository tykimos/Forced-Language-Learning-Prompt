# Language Echo Rule (Italian → Portuguese)

> For **Italian** (Italiano) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Italian.
