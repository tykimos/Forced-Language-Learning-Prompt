# Language Echo Rule (German → Thai)

> For **German** (Deutsch) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in German.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in German.
