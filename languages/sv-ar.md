# Language Echo Rule (Swedish → Arabic)

> For **Swedish** (Svenska) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Swedish.
