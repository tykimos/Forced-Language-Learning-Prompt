# Language Echo Rule (Filipino → Thai)

> For **Filipino** (Filipino) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Filipino.
