# Language Echo Rule (Indonesian → Turkish)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Indonesian.
