# Language Echo Rule (Dutch → Malay)

> For **Dutch** (Nederlands) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Dutch.
