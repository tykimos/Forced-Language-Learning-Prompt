# Language Echo Rule (Thai → Swedish)

> For **Thai** (ภาษาไทย) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Thai.
