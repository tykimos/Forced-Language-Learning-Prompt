# Language Echo Rule (Ukrainian → Indonesian)

> For **Ukrainian** (Українська) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Ukrainian.
