# Language Echo Rule (Vietnamese → Russian)

> For **Vietnamese** (Tiếng Việt) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Vietnamese.
