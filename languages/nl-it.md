# Language Echo Rule (Dutch → Italian)

> For **Dutch** (Nederlands) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Dutch.
