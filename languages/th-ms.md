# Language Echo Rule (Thai → Malay)

> For **Thai** (ภาษาไทย) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Thai.
