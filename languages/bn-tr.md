# Language Echo Rule (Bengali → Turkish)

> For **Bengali** (বাংলা) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Bengali.
