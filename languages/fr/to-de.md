# Language Echo Rule (French → German)

> For **French** (Français) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct German sentence, then proceed with the task in French.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in French.
