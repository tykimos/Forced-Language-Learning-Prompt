# Language Echo Rule (French → Portuguese)

> For **French** (Français) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in French.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in French.
