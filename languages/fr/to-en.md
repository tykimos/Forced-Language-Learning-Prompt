# Language Echo Rule (French → English)

> For **French** (Français) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct English sentence, then proceed with the task in French.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in French.
