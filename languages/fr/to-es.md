# Language Echo Rule (French → Spanish)

> For **French** (Français) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in French.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in French.
