# Language Echo Rule (German → Portuguese)

> For **German** (Deutsch) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in German.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in German.
