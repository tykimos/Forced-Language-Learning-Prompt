# Language Echo Rule (Polish → Malay)

> For **Polish** (Polski) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Polish.
