# Language Echo Rule (Russian → French)

> For **Russian** (Русский) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Russian.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Russian.
