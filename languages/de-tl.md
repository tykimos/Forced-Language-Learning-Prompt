# Language Echo Rule (German → Filipino)

> For **German** (Deutsch) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in German.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in German.
