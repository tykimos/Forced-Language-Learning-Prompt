# Language Echo Rule (Japanese → Arabic)

> For **Japanese** (日本語) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Japanese.
