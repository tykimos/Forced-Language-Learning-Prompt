# Language Echo Rule (Italian → Filipino)

> For **Italian** (Italiano) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Italian.
