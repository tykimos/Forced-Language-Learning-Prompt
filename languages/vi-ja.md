# Language Echo Rule (Vietnamese → Japanese)

> For **Vietnamese** (Tiếng Việt) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Vietnamese.
