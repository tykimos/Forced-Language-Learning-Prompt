# Language Echo Rule (Thai → Vietnamese)

> For **Thai** (ภาษาไทย) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Thai.
