# Language Echo Rule (Russian → Italian)

> For **Russian** (Русский) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Russian.
