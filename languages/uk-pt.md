# Language Echo Rule (Ukrainian → Portuguese)

> For **Ukrainian** (Українська) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Ukrainian.
