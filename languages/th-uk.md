# Language Echo Rule (Thai → Ukrainian)

> For **Thai** (ภาษาไทย) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Thai.
