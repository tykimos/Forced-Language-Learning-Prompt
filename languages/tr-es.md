# Language Echo Rule (Turkish → Spanish)

> For **Turkish** (Türkçe) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Turkish.
