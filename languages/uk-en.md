# Language Echo Rule (Ukrainian → English)

> For **Ukrainian** (Українська) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Ukrainian.
