# Language Echo Rule (Polish → German)

> For **Polish** (Polski) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Polish.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Polish.
