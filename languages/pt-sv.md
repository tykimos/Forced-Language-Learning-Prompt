# Language Echo Rule (Portuguese → Swedish)

> For **Portuguese** (Português) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Portuguese.
