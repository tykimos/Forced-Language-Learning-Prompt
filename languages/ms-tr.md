# Language Echo Rule (Malay → Turkish)

> For **Malay** (Bahasa Melayu) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Malay.
