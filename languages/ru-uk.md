# Language Echo Rule (Russian → Ukrainian)

> For **Russian** (Русский) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Russian.
