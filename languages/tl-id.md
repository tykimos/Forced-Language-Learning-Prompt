# Language Echo Rule (Filipino → Indonesian)

> For **Filipino** (Filipino) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Filipino.
