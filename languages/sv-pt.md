# Language Echo Rule (Swedish → Portuguese)

> For **Swedish** (Svenska) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Swedish.
