# Language Echo Rule (Chinese → Portuguese)

> For **Chinese** (中文) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Chinese.
