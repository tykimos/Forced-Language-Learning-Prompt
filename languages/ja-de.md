# Language Echo Rule (Japanese → German)

> For **Japanese** (日本語) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Japanese.
