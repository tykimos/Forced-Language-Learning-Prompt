# Language Echo Rule (Turkish → Thai)

> For **Turkish** (Türkçe) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Turkish.
