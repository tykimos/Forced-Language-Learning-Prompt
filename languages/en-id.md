# Language Echo Rule (English → Indonesian)

> For **English** (English) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in English.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in English.
