# Language Echo Rule (Hindi → Dutch)

> For **Hindi** (हिन्दी) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Hindi.
