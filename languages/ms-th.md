# Language Echo Rule (Malay → Thai)

> For **Malay** (Bahasa Melayu) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Malay.
