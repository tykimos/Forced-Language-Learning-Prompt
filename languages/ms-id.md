# Language Echo Rule (Malay → Indonesian)

> For **Malay** (Bahasa Melayu) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Malay.
