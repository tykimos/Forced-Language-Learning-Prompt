# Language Echo Rule (Thai → Russian)

> For **Thai** (ภาษาไทย) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Thai.
