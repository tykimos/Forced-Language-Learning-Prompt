# Language Echo Rule (French → Malay)

> For **French** (Français) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in French.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in French.
