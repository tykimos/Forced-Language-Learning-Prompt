# Language Echo Rule (English → Bengali)

> For **English** (English) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in English.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in English.
