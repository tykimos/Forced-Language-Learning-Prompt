# Language Echo Rule (Portuguese → Dutch)

> For **Portuguese** (Português) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Portuguese.
