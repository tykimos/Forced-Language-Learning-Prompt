# Language Echo Rule (Polish → French)

> For **Polish** (Polski) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Polish.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Polish.
