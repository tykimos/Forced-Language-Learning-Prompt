# Language Echo Rule (Ukrainian → Malay)

> For **Ukrainian** (Українська) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Ukrainian.
