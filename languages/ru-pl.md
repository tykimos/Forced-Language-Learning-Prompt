# Language Echo Rule (Russian → Polish)

> For **Russian** (Русский) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Russian.
