# Language Echo Rule (Hindi → French)

> For **Hindi** (हिन्दी) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Hindi.
