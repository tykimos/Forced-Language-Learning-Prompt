# Language Echo Rule (Thai → Portuguese)

> For **Thai** (ภาษาไทย) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Thai.
