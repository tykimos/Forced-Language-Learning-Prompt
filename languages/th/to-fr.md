# Language Echo Rule (Thai → French)

> For **Thai** (ภาษาไทย) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Thai.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Thai.
