# Language Echo Rule (Thai → Chinese)

> For **Thai** (ภาษาไทย) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Thai.
