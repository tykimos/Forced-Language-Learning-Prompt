# Language Echo Rule (Italian → Bengali)

> For **Italian** (Italiano) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Italian.
