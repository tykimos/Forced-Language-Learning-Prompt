# Language Echo Rule (French → Swedish)

> For **French** (Français) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in French.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in French.
