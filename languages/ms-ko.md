# Language Echo Rule (Malay → Korean)

> For **Malay** (Bahasa Melayu) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Malay.
