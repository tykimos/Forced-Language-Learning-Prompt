# Language Echo Rule (Turkish → Portuguese)

> For **Turkish** (Türkçe) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Turkish.
