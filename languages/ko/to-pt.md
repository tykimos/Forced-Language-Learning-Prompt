# Language Echo Rule (Korean → Portuguese)

> For **Korean** (한국어) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Korean.
