# Language Echo Rule (Korean → French)

> For **Korean** (한국어) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Korean.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Korean.
