# Language Echo Rule (Korean → Chinese)

> For **Korean** (한국어) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Korean.
