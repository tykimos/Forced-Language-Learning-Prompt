# Language Echo Rule (Korean → Spanish)

> For **Korean** (한국어) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Korean.
