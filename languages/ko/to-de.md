# Language Echo Rule (Korean → German)

> For **Korean** (한국어) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Korean.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Korean.
