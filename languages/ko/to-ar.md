# Language Echo Rule (Korean → Arabic)

> For **Korean** (한국어) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Korean.
