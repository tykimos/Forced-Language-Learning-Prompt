# Language Echo Rule (Korean → Japanese)

> For **Korean** (한국어) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Korean.
