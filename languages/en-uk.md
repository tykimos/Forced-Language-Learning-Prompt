# Language Echo Rule (English → Ukrainian)

> For **English** (English) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in English.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in English.
