# Language Echo Rule (Malay → Filipino)

> For **Malay** (Bahasa Melayu) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Malay.
