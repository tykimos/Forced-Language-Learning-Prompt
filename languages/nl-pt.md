# Language Echo Rule (Dutch → Portuguese)

> For **Dutch** (Nederlands) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Dutch.
