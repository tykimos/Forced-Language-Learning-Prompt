# Language Echo Rule (Vietnamese → Arabic)

> For **Vietnamese** (Tiếng Việt) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Vietnamese.
