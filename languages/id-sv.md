# Language Echo Rule (Indonesian → Swedish)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Indonesian.
