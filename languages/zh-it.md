# Language Echo Rule (Chinese → Italian)

> For **Chinese** (中文) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Chinese.
