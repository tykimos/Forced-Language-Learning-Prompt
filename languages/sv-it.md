# Language Echo Rule (Swedish → Italian)

> For **Swedish** (Svenska) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Swedish.
