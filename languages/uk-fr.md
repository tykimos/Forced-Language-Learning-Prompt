# Language Echo Rule (Ukrainian → French)

> For **Ukrainian** (Українська) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Ukrainian.
