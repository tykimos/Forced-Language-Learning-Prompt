# Language Echo Rule (Japanese → Chinese)

> For **Japanese** (日本語) speakers learning **Chinese** (中文)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Chinese sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Chinese**, first show a refined/corrected version of the Chinese sentence, then proceed with the task in Japanese.
