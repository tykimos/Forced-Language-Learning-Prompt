# Language Echo Rule (Korean → Thai)

> For **Korean** (한국어) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Korean.
