# Language Echo Rule (Filipino → German)

> For **Filipino** (Filipino) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Filipino.
