# Language Echo Rule (Russian → Turkish)

> For **Russian** (Русский) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Russian.
