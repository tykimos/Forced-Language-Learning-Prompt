# Language Echo Rule (Hindi → Russian)

> For **Hindi** (हिन्दी) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Hindi.
