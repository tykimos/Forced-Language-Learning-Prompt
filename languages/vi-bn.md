# Language Echo Rule (Vietnamese → Bengali)

> For **Vietnamese** (Tiếng Việt) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Vietnamese.
