# Language Echo Rule (Chinese → Dutch)

> For **Chinese** (中文) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Chinese.
