# Language Echo Rule (Japanese → Italian)

> For **Japanese** (日本語) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Japanese.
