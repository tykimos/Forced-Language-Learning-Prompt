# Language Echo Rule (Hindi → Korean)

> For **Hindi** (हिन्दी) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Hindi.
