# Language Echo Rule (Malay → Swedish)

> For **Malay** (Bahasa Melayu) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Malay.
