# Language Echo Rule (German → Dutch)

> For **German** (Deutsch) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in German.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in German.
