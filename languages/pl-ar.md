# Language Echo Rule (Polish → Arabic)

> For **Polish** (Polski) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Polish.
