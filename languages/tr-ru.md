# Language Echo Rule (Turkish → Russian)

> For **Turkish** (Türkçe) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Turkish.
