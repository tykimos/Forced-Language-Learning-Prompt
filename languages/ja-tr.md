# Language Echo Rule (Japanese → Turkish)

> For **Japanese** (日本語) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Japanese.
