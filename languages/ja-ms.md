# Language Echo Rule (Japanese → Malay)

> For **Japanese** (日本語) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Japanese.
