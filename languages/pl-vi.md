# Language Echo Rule (Polish → Vietnamese)

> For **Polish** (Polski) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Polish.
