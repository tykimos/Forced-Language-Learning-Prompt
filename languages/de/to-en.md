# Language Echo Rule (German → English)

> For **German** (Deutsch) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct English sentence, then proceed with the task in German.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in German.
