# Language Echo Rule (German → French)

> For **German** (Deutsch) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct French sentence, then proceed with the task in German.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in German.
