# Language Echo Rule (German → Spanish)

> For **German** (Deutsch) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in German.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in German.
