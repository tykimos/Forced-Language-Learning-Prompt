# Language Echo Rule (German → Vietnamese)

> For **German** (Deutsch) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in German.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in German.
