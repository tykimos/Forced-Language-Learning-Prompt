# Language Echo Rule (Polish → Turkish)

> For **Polish** (Polski) speakers learning **Turkish** (Türkçe)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Turkish sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Turkish**, first show a refined/corrected version of the Turkish sentence, then proceed with the task in Polish.
