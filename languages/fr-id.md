# Language Echo Rule (French → Indonesian)

> For **French** (Français) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in French.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in French.
