# Language Echo Rule (French → Bengali)

> For **French** (Français) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in French.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in French.
