# Language Echo Rule (Arabic → Swedish)

> For **Arabic** (العربية) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Arabic.
