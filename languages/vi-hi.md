# Language Echo Rule (Vietnamese → Hindi)

> For **Vietnamese** (Tiếng Việt) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Vietnamese.
