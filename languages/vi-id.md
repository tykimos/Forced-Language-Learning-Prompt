# Language Echo Rule (Vietnamese → Indonesian)

> For **Vietnamese** (Tiếng Việt) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Vietnamese.
