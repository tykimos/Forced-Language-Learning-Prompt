# Language Echo Rule (Hindi → Portuguese)

> For **Hindi** (हिन्दी) speakers learning **Portuguese** (Português)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Portuguese sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Portuguese**, first show a refined/corrected version of the Portuguese sentence, then proceed with the task in Hindi.
