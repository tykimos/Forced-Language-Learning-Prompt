# Language Echo Rule (Portuguese → Korean)

> For **Portuguese** (Português) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Portuguese.
