# Language Echo Rule (Filipino → Italian)

> For **Filipino** (Filipino) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Filipino.
