# Language Echo Rule (Chinese → Vietnamese)

> For **Chinese** (中文) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Chinese.
