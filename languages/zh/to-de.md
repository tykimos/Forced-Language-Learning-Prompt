# Language Echo Rule (Chinese → German)

> For **Chinese** (中文) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Chinese.
