# Language Echo Rule (Chinese → French)

> For **Chinese** (中文) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Chinese.
