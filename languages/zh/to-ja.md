# Language Echo Rule (Chinese → Japanese)

> For **Chinese** (中文) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Chinese.
