# Language Echo Rule (Chinese → Korean)

> For **Chinese** (中文) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Chinese.
