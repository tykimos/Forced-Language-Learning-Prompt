# Language Echo Rule (Polish → Ukrainian)

> For **Polish** (Polski) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Polish.
