# Language Echo Rule (Japanese → French)

> For **Japanese** (日本語) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Japanese.
