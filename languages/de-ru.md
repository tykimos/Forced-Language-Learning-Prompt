# Language Echo Rule (German → Russian)

> For **German** (Deutsch) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **German**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in German.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in German.
