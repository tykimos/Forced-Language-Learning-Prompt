# Language Echo Rule (Russian → Swedish)

> For **Russian** (Русский) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Russian**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Russian.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Russian.
