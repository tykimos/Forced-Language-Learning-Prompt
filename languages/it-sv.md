# Language Echo Rule (Italian → Swedish)

> For **Italian** (Italiano) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Italian**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Italian.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Italian.
