# Language Echo Rule (English → Italian)

> For **English** (English) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in English.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in English.
