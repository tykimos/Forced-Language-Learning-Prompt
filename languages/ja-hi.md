# Language Echo Rule (Japanese → Hindi)

> For **Japanese** (日本語) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Japanese.
