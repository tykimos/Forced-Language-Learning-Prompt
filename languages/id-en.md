# Language Echo Rule (Indonesian → English)

> For **Indonesian** (Bahasa Indonesia) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Indonesian.
