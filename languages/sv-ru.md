# Language Echo Rule (Swedish → Russian)

> For **Swedish** (Svenska) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Swedish.
