# Language Echo Rule (Korean → Ukrainian)

> For **Korean** (한국어) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Korean.
