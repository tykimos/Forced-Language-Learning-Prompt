# Language Echo Rule (Swedish → Thai)

> For **Swedish** (Svenska) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Swedish.
