# Language Echo Rule (Portuguese → French)

> For **Portuguese** (Português) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Portuguese.
