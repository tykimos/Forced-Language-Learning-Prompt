# Language Echo Rule (Chinese → Hindi)

> For **Chinese** (中文) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Chinese.
