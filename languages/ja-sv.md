# Language Echo Rule (Japanese → Swedish)

> For **Japanese** (日本語) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Japanese.
