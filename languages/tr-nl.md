# Language Echo Rule (Turkish → Dutch)

> For **Turkish** (Türkçe) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Turkish.
