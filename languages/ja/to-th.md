# Language Echo Rule (Japanese → Thai)

> For **Japanese** (日本語) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Japanese.
