# Language Echo Rule (Japanese → Spanish)

> For **Japanese** (日本語) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Japanese.
