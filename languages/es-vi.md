# Language Echo Rule (Spanish → Vietnamese)

> For **Spanish** (Español) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Spanish**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Spanish.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Spanish.
