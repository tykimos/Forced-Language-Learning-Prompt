# Language Echo Rule (Korean → Hindi)

> For **Korean** (한국어) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Korean**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Korean.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Korean.
