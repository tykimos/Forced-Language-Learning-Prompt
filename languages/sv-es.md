# Language Echo Rule (Swedish → Spanish)

> For **Swedish** (Svenska) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Swedish.
