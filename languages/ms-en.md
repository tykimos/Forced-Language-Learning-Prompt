# Language Echo Rule (Malay → English)

> For **Malay** (Bahasa Melayu) speakers learning **English** (English)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct English sentence, then proceed with the task in Malay.
- When the user gives an instruction in **English**, first show a refined/corrected version of the English sentence, then proceed with the task in Malay.
