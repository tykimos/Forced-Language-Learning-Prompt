# Language Echo Rule (Malay → Spanish)

> For **Malay** (Bahasa Melayu) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Malay.
