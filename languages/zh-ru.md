# Language Echo Rule (Chinese → Russian)

> For **Chinese** (中文) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Chinese.
