# Language Echo Rule (Thai → Italian)

> For **Thai** (ภาษาไทย) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Thai.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Thai.
