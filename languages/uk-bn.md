# Language Echo Rule (Ukrainian → Bengali)

> For **Ukrainian** (Українська) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Ukrainian**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Ukrainian.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Ukrainian.
