# Language Echo Rule (Hindi → Swedish)

> For **Hindi** (हिन्दी) speakers learning **Swedish** (Svenska)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Swedish sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Swedish**, first show a refined/corrected version of the Swedish sentence, then proceed with the task in Hindi.
