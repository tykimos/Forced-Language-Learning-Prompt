# Language Echo Rule (Indonesian → Vietnamese)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in Indonesian.
