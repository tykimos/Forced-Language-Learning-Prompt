# Language Echo Rule (Indonesian → Korean)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Indonesian.
