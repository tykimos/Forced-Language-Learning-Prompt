# Language Echo Rule (Indonesian → Hindi)

> For **Indonesian** (Bahasa Indonesia) speakers learning **Hindi** (हिन्दी)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct Hindi sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **Hindi**, first show a refined/corrected version of the Hindi sentence, then proceed with the task in Indonesian.
