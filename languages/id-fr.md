# Language Echo Rule (Indonesian → French)

> For **Indonesian** (Bahasa Indonesia) speakers learning **French** (Français)

## Language Echo Rule
- When the user gives an instruction in **Indonesian**, first restate the instruction as a clear, correct French sentence, then proceed with the task in Indonesian.
- When the user gives an instruction in **French**, first show a refined/corrected version of the French sentence, then proceed with the task in Indonesian.
