# Language Echo Rule (Arabic → Ukrainian)

> For **Arabic** (العربية) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Arabic.
