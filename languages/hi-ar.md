# Language Echo Rule (Hindi → Arabic)

> For **Hindi** (हिन्दी) speakers learning **Arabic** (العربية)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Arabic sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Arabic**, first show a refined/corrected version of the Arabic sentence, then proceed with the task in Hindi.
