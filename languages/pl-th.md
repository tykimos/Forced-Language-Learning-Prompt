# Language Echo Rule (Polish → Thai)

> For **Polish** (Polski) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Polish**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Polish.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Polish.
