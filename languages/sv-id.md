# Language Echo Rule (Swedish → Indonesian)

> For **Swedish** (Svenska) speakers learning **Indonesian** (Bahasa Indonesia)

## Language Echo Rule
- When the user gives an instruction in **Swedish**, first restate the instruction as a clear, correct Indonesian sentence, then proceed with the task in Swedish.
- When the user gives an instruction in **Indonesian**, first show a refined/corrected version of the Indonesian sentence, then proceed with the task in Swedish.
