# Language Echo Rule (Hindi → Filipino)

> For **Hindi** (हिन्दी) speakers learning **Filipino** (Filipino)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Filipino sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Filipino**, first show a refined/corrected version of the Filipino sentence, then proceed with the task in Hindi.
