# Language Echo Rule (Chinese → Polish)

> For **Chinese** (中文) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Chinese.
