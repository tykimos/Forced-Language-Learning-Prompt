# Language Echo Rule (Vietnamese → Ukrainian)

> For **Vietnamese** (Tiếng Việt) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Vietnamese**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Vietnamese.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Vietnamese.
