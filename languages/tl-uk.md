# Language Echo Rule (Filipino → Ukrainian)

> For **Filipino** (Filipino) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Filipino.
