# Language Echo Rule (Portuguese → Bengali)

> For **Portuguese** (Português) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Portuguese**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Portuguese.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Portuguese.
