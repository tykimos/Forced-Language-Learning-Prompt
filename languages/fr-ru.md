# Language Echo Rule (French → Russian)

> For **French** (Français) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **French**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in French.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in French.
