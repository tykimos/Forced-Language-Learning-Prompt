# Language Echo Rule (Thai → German)

> For **Thai** (ภาษาไทย) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Thai**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Thai.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Thai.
