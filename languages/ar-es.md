# Language Echo Rule (Arabic → Spanish)

> For **Arabic** (العربية) speakers learning **Spanish** (Español)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Spanish sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Spanish**, first show a refined/corrected version of the Spanish sentence, then proceed with the task in Arabic.
