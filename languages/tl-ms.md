# Language Echo Rule (Filipino → Malay)

> For **Filipino** (Filipino) speakers learning **Malay** (Bahasa Melayu)

## Language Echo Rule
- When the user gives an instruction in **Filipino**, first restate the instruction as a clear, correct Malay sentence, then proceed with the task in Filipino.
- When the user gives an instruction in **Malay**, first show a refined/corrected version of the Malay sentence, then proceed with the task in Filipino.
