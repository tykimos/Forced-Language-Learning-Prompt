# Language Echo Rule (Dutch → German)

> For **Dutch** (Nederlands) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Dutch**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Dutch.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Dutch.
