# Language Echo Rule (Arabic → Thai)

> For **Arabic** (العربية) speakers learning **Thai** (ภาษาไทย)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Thai sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Thai**, first show a refined/corrected version of the Thai sentence, then proceed with the task in Arabic.
