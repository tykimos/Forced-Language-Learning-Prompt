# Language Echo Rule (Arabic → German)

> For **Arabic** (العربية) speakers learning **German** (Deutsch)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct German sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **German**, first show a refined/corrected version of the German sentence, then proceed with the task in Arabic.
