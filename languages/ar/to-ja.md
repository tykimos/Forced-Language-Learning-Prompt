# Language Echo Rule (Arabic → Japanese)

> For **Arabic** (العربية) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Arabic**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Arabic.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Arabic.
