# Language Echo Rule (Japanese → Korean)

> For **Japanese** (日本語) speakers learning **Korean** (한국어)

## Language Echo Rule
- When the user gives an instruction in **Japanese**, first restate the instruction as a clear, correct Korean sentence, then proceed with the task in Japanese.
- When the user gives an instruction in **Korean**, first show a refined/corrected version of the Korean sentence, then proceed with the task in Japanese.
