# Language Echo Rule (Malay → Dutch)

> For **Malay** (Bahasa Melayu) speakers learning **Dutch** (Nederlands)

## Language Echo Rule
- When the user gives an instruction in **Malay**, first restate the instruction as a clear, correct Dutch sentence, then proceed with the task in Malay.
- When the user gives an instruction in **Dutch**, first show a refined/corrected version of the Dutch sentence, then proceed with the task in Malay.
