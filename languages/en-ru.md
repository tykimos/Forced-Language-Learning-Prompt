# Language Echo Rule (English → Russian)

> For **English** (English) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in English.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in English.
