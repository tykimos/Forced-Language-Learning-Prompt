# Language Echo Rule (Hindi → Italian)

> For **Hindi** (हिन्दी) speakers learning **Italian** (Italiano)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Italian sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Italian**, first show a refined/corrected version of the Italian sentence, then proceed with the task in Hindi.
