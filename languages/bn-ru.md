# Language Echo Rule (Bengali → Russian)

> For **Bengali** (বাংলা) speakers learning **Russian** (Русский)

## Language Echo Rule
- When the user gives an instruction in **Bengali**, first restate the instruction as a clear, correct Russian sentence, then proceed with the task in Bengali.
- When the user gives an instruction in **Russian**, first show a refined/corrected version of the Russian sentence, then proceed with the task in Bengali.
