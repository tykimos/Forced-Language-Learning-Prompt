# Language Echo Rule (English → Vietnamese)

> For **English** (English) speakers learning **Vietnamese** (Tiếng Việt)

## Language Echo Rule
- When the user gives an instruction in **English**, first restate the instruction as a clear, correct Vietnamese sentence, then proceed with the task in English.
- When the user gives an instruction in **Vietnamese**, first show a refined/corrected version of the Vietnamese sentence, then proceed with the task in English.
