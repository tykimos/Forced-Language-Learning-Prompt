# Language Echo Rule (Turkish → Japanese)

> For **Turkish** (Türkçe) speakers learning **Japanese** (日本語)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Japanese sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Japanese**, first show a refined/corrected version of the Japanese sentence, then proceed with the task in Turkish.
