# Language Echo Rule (Turkish → Polish)

> For **Turkish** (Türkçe) speakers learning **Polish** (Polski)

## Language Echo Rule
- When the user gives an instruction in **Turkish**, first restate the instruction as a clear, correct Polish sentence, then proceed with the task in Turkish.
- When the user gives an instruction in **Polish**, first show a refined/corrected version of the Polish sentence, then proceed with the task in Turkish.
