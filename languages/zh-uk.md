# Language Echo Rule (Chinese → Ukrainian)

> For **Chinese** (中文) speakers learning **Ukrainian** (Українська)

## Language Echo Rule
- When the user gives an instruction in **Chinese**, first restate the instruction as a clear, correct Ukrainian sentence, then proceed with the task in Chinese.
- When the user gives an instruction in **Ukrainian**, first show a refined/corrected version of the Ukrainian sentence, then proceed with the task in Chinese.
