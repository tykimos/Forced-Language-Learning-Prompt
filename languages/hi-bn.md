# Language Echo Rule (Hindi → Bengali)

> For **Hindi** (हिन्दी) speakers learning **Bengali** (বাংলা)

## Language Echo Rule
- When the user gives an instruction in **Hindi**, first restate the instruction as a clear, correct Bengali sentence, then proceed with the task in Hindi.
- When the user gives an instruction in **Bengali**, first show a refined/corrected version of the Bengali sentence, then proceed with the task in Hindi.
